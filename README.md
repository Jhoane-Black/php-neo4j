# php-neo4j

Connecting PHP to Neo4j

Neo4j offers us a large number of clients to interact with it through different programming languages (Neo4j Drivers).
In our example we will interact with Neo4j through PHP. For PHP there are several drivers (Neo4jPHP, NeoClient, ...). In this tutorial we will use GraphAware PHP Client

Installation üöÄ

As a previous step to the installation of GraphAware PHP Client we must have Neo4j,Apache Server,PHP and Composer installed.
In the project folder we have to create a file composer.json in which we specify the dependency with GraphAware PHP Client.

{
  "require": {
  "graphaware/neo4j-php-client": "4.0"
  }
}

Next, in the project folder we will install the dependencies by opening the cmd or powershell in this folder and typing the following command 

composer install 

Connection to Neo4j 

Before proceeding to create the connection we have to import the autoload.php file generated by Composer when installing the dependencies.
The connection will be created using the create() method of the GraphAware PHP ClientBuilder class. In it we will indicate several parameters, highlighting the host, password and port. In our case we have used localhost and 7474 (the default port of Neo4j). 

<?php
require_once "vendor/autoload.php";
use GraphAware\Neo4j\Client\ClientBuilder;
$client = ClientBuilder::create()
->addConnection("default", "http://neo4j:12345@localhost:7474")
//Example for HTTP connection configuration (port is optional)
->addConnection("bolt", "bolt://neo4j:12345@localhost:7687")
// Example for BOLT connection configuration (port is optional)
->build();
$client->run('CREATE (n:book{title:"Cien A√±os de Soledad‚Ñ¢: Gabo"})');
?>
